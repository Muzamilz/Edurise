# Edurise Platform Documentation

This directory contains all documentation for the Edurise educational platform.

├── ZOOM_API_SETUP.md                  # Zoom API integration guide

## Quick Start

1. **Setup**: Start with `REPOSITORY_SETUP.md` for initial project setup
2. **Data**: Follow `DATA_SETUP_README.md` for database configuration
3. **Redis**: Configure Redis using `REDIS_SETUP.md`
4. **Zoom**: Set up Zoom integration with `ZOOM_API_SETUP.md`
